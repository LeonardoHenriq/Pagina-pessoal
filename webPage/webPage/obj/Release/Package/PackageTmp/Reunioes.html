Reuniões/Agendas.

Reuniões/Agendas se trata de um sistema web para agendar reniões nas salas da empresa Transreal e uma lista de contatos com email e ramal dos funcionários.

Vamos falar um pouco sobre o sistema

Nos parágrafos abaixo vou resumir um pouco em algumas sessões informações sobre o processo de criação do projeto
favorite

Motivação

Primeiramente gostaria de agradecer novamente a confiança de todo time transreal e aos envolvidos no projeto.
O Reuniões/Agendas surgiu para facilitar a utilização das salas de reuniões.
E foi imposto a minha pessoa a responsabilidade da criação do mesmo!

build

Tecnologias Utilizadas

Para o desenvolvimento do sistema inicialmente utilizei a ferramenta Adobe XD para elaboração dos layouts de interface para aprovação dos usuários.
Após a aprovação dos layouts e levantamento dos requisitos, comecei a desenvolver a aplicação, para o front end utilizei o framework Angular e a biblioteca Angular Material. E para o backend construi uma API rest com alguns EndPoints que são consumidos pelo frontend com ASP.NET Core MVC e Entity Framework. E para o banco dados utilizei o SQL Server.

settings_suggest

Funcionalidades

A principal funcionalidade do Reuniões/Agendas é agendar reuniões em algumas das salas que a empresa possui, assim os funcionários podem verificar se a sala está sendo utilizada ou não além de facilitar na hora da configuração da sala para a equipe de Ti.
Além disso o sistema também possui a lista de contatos que contém informações de ramais e emails dos funcionários da empresa.
Após o agendamento da reunião ela aparece na tabela de visualização das reuniões marcadas com o status de agendada, o status é atualizado em tempo real conforme o periodo em que a reunião foi marcada!
